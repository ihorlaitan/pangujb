Jailbreak for iOS 9.2 ~ 9.3

I made this simple page on GitHub because other websites couldn't handle so much install requests.

Install Jailbreak

Instructions

      Install the app (Press the button above)
      Trust the certificate (Settings > General > Devices Management)
      Open the app, allow it to send notifications
      Uncheck the checkbox option, press the circle on the middle
      Lock the screen, you'll receive a notification
      Wait the device reboot, then check if you have Cydia

Requirements

      iOS 9.2, 9.2.1, 9.3, 9.3.1, 9.3.2, 9.3.3
      iPhone 5s, 6, 6+, 6s, 6s+, SE
      iPod Touch 6G
      iPad Mini 2, 3, 4
      iPad Air 1, 2
      iPad Pro (both sizes)
